# Examples

## Overview

This project includes runnable examples under the `examples/` directory when present. Examples demonstrate common usage patterns and integration scenarios.

## Finding Examples

```bash
ls examples/
```

## Running Examples

Each example contains its own README with setup and run instructions. Most examples assume the workspace has been cloned recursively:

```bash
git clone --recursive https://github.com/starlink-awaken/omostation.git
cd omostation/projects/metaos/examples
```

## Requesting New Examples

If an example you need is missing, open an issue with a description of the use case.
