# MetaOS API / Usage Reference

> Quick reference for using **MetaOS** programmatically and from the command line.

## Command Line

- `metaos --help` — CLI
- `metaos status` — system health
- `metaos-mcp` — MCP server

## Programmatic API

Import `metaos` or use the CLI/MCP interfaces.

## Configuration

- Stack: python
- Dependencies: see [`../pyproject.toml`](../pyproject.toml) (Python) or [`../package.json`](../package.json) (TypeScript).
- Environment variables and ports: see workspace `protocols/port-registry.yaml` and root `.env.example`.
