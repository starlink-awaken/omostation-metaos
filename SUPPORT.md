# Support

## Getting Help

- Read the [FAQ](docs/FAQ.md) and [Troubleshooting Guide](docs/TROUBLESHOOTING.md).
- Review the [API / Usage Reference](docs/API.md) and [Architecture Overview](docs/ARCHITECTURE.md).

## Asking Questions

1. Search existing [GitHub Discussions](https://github.com/starlink-awaken/omostation/discussions) and issues.
2. Open a new discussion for usage questions.
3. Open an issue for bugs or feature requests with a minimal reproducible example.

## Commercial Support

For commercial support inquiries, contact the omostation maintainers via the workspace root channels.
